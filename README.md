- 🧩 图文并茂
- 🔨 提供 docker image

## 安装 📦

```shell
docker run -itd --name sb3-study-docs -p 80:80 ghcr.io/mcdd1024/sb3-study-docs:version
```

> [!TIP]
>
> 注意将命令中的 `version` 替换为相应版本 https://github.com/mcdd1024/sb3-study/pkgs/container/sb3-study-docs
